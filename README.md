# 📖 My Learning Journal

A responsive multi-page blog-style website built with HTML and CSS that showcases a personal learning journey as a frontend developer. The project includes a homepage, blog post previews, and navigation between pages.

---

## ✨ Features

- Multi-page website (Home, About, Blog pages)
- Responsive layout for desktop and mobile
- Blog post preview grid
- Hero section highlighting latest post
- Clean navigation header and footer
- Custom typography using Google Fonts
- Accessible image alt text for better usability

---

## 🛠️ Built With

- HTML5
- CSS3
- Google Fonts
- Responsive Design Principles

To run this project locally:

1. Clone the repository:
```bash

```
2. Open the project folder
3. Open index.html in your browser
